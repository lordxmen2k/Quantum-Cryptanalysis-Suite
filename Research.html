Abstract

We present the Quantum Cryptanalysis Suite (QCS), a pedagogical framework implementing exact state-vector simulation of three canonical quantum algorithms applied to toy cryptographic instances. The suite comprises (1) the Quantum Approximate Optimization Algorithm (QAOA) for lattice problems including the Shortest Vector Problem (SVP) and Learning With Errors (LWE), (2) Shor's period-finding algorithm for integer factorization on composite numbers up to N = 35, and (3) Grover's amplitude amplification for symmetric key search on 4–6 bit toy ciphers. All algorithms execute on a lightweight, gate-level state-vector simulator supporting up to ~16 qubits on consumer hardware. We provide rigorous architectural documentation, experimental validation, and explicit mappings from heuristic concepts to their corresponding quantum algorithmic primitives. This work serves as an educational reference for understanding the mechanics of quantum cryptanalysis algorithms through executable simulation.

Introduction

The intersection of quantum computing and cryptanalysis has produced some of the most consequential theoretical results in modern computer science. Shor's algorithm (1994) demonstrated that a universal quantum computer could factor integers and compute discrete logarithms in polynomial time, threatening the foundations of RSA and Elliptic Curve Cryptography (ECC). Grover's algorithm (1996) established a quadratic speedup for unstructured search, effectively halving the security margin of symmetric ciphers. More recently, the Quantum Approximate Optimization Algorithm (QAOA) has emerged as a leading candidate for solving combinatorial optimization problems on near-term quantum hardware, with direct applications to lattice-based cryptanalysis.

However, the growing public interest in quantum computing has also produced a proliferation of quantum-washing — the presentation of classical heuristics in quantum terminology without genuine quantum mechanical content. This document provides a rigorous alternative: a complete, executable suite of actual quantum algorithms running on an exact state-vector simulator, with explicit architectural documentation and experimental validation. All code is provided under the Apache 2.0 license for educational and research use.

System Architecture

# State-Vector Simulator

The computational backend is a lightweight, gate-level quantum circuit simulator that stores the full 2n complex amplitude vector and applies unitary operations via direct matrix-vector multiplication. The simulator supports single-qubit gates (H, X, Y, Z, RX, RY, RZ), two-qubit gates (CNOT, CZ, controlled-phase, SWAP), and multi-qubit primitives including the Quantum Fourier Transform (QFT) and its inverse. On consumer hardware, the simulator comfortably handles up to approximately 16 qubits (65,536 amplitudes), beyond which memory requirements grow exponentially.

Python 
class QuantumSimulator:
    def __init__(self, n_qubits: int):
        self.n = n_qubits
        self.N = 1 << n_qubits          # 2^n amplitudes
        self.state = np.zeros(self.N, dtype=complex)
        self.state[0] = 1.0              # |0...0⟩

    def h(self, q: int):               # Hadamard gate
        stride = 1 << q
        for base in range(0, self.N, stride << 1):
            for j in range(base, base + stride):
                a, b = self.state[j], self.state[j + stride]
                self.state[j] = (a + b) / np.sqrt(2)
                self.state[j + stride] = (a - b) / np.sqrt(2)

# QAOA for Lattice Cryptography

The Quantum Approximate Optimization Algorithm (Farhi et al., 2014) is applied to two fundamental lattice problems: the Shortest Vector Problem (SVP) and Learning With Errors (LWE). Both problems are encoded as diagonal cost Hamiltonians where the ground state corresponds to the optimal cryptanalytic solution.

For SVP-2D, integer coefficients (s, t) are encoded in qubit registers spanning [−3, +3] in each dimension. The cost function computes the squared norm ‖s·b₁ + t·b₂‖² with a heavy penalty for the zero vector. For LWE, secret variables sᵢ ∈ ℤq are encoded in ⌈log₂(q)⌉ qubits each, and the cost measures the squared distance between A·s and the target vector b modulo q.

Python 
def _apply_cost(self, gamma: float):
    # Apply e^(-i * gamma * H_C) where H_C is diagonal
    phases = np.exp(-1j * gamma * self._cost_array)
    self.sim.state *= phases

def _circuit(self, params: np.ndarray):
    self.sim.reset()
    self.sim.h_all()                  # Uniform superposition
    for layer in range(self.p):
        self._apply_cost(gammas[layer]) # Cost evolution
        for q in range(self.n):
            self.sim.rx(2 * betas[layer], q)  # Mixer evolution

# Shor's Algorithm for Integer Factorization

Shor's algorithm (Shor, 1994) reduces integer factorization to order-finding via quantum period detection. The implementation follows the standard three-register architecture: a work register initialized in uniform superposition, an auxiliary register initialized to |1⟩, and a modular exponentiation oracle that computes ax mod N reversibly. The Quantum Fourier Transform on the work register extracts the period r, from which factors are recovered via gcd(ar/2 ± 1, N).

# Shor's core: period finding via QFT
for q in range(n_work):

# Grover's Algorithm for Symmetric Key Search

Grover's algorithm (Grover, 1996) provides a quadratic speedup for unstructured search problems. For an n-bit key space, the optimal number of iterations is ⌊π/4 · √N⌋, where N = 2n. The algorithm alternates between a phase oracle (which marks the correct key by flipping its phase) and a diffusion operator (which amplifies the amplitude of marked states).

Two toy ciphers are provided: a 4-bit XOR+S-box cipher and a 6-bit 2-round Feistel network. In both cases, Grover recovers the key with probability approaching the theoretical maximum (~96% for 4 bits with optimal iterations).
